# Scope crm_tsm

## Цель

Запустить локальный пилот системы управления задачами вместо Bitrix-задач.

## Закрываемые процессы

- фиксация задач;
- сроки выполнения;
- назначение ответственных;
- разделение по проектам;
- роли и права доступа;
- вложения файлов в задачи;
- напоминания;
- календарь;
- будущая интеграция с Telegram-группой монтажников.

## Что не делаем в первом этапе

- полноценную CRM продаж;
- бухгалтерию;
- склад;
- HR;
- production-деплой;
- интеграцию с 1C/Wialon/Arvento.

## Критерии успеха пилота

- задача создаётся за 1–2 минуты;
- у задачи есть проект, срок, ответственный и вложения;
- руководитель видит просроченные задачи;
- монтажник получает понятное задание;
- статус задачи можно быстро обновить;
- структура прав не даёт монтажникам видеть лишние проекты.
