# Модель проектов

## Базовая структура

Проект = направление, объект, участок или крупная зона ответственности.

Примеры:

- Монтаж GPS
- Сервис GPS
- АЗС / Turpak
- Пассажирский транспорт
- СКУД
- Тестовый проект

## Рекомендованная структура задачи

- Проект
- Тип задачи
- Название
- Описание
- Ответственный
- Срок
- Приоритет
- Статус
- Вложения
- Комментарии

## Правило наименования

Формат названия:

```text
[Объект/машина] Краткое действие
```

Примеры:

```text
[34ABC123] Установить GPS-трекер
[АЗС-01] Проверить связь Turpak
[Автобус 12] Проверить валидатор
```
