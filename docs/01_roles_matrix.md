# Ролевая модель

| Роль | Назначение | Права в пилоте |
|---|---|---|
| Администратор | Техническая настройка системы | Полный доступ |
| Руководитель проекта | Управление проектом/объектом | Создание задач, назначение, сроки, закрытие, отчёты |
| Координатор | Диспетчеризация задач | Создание задач, изменение статусов, комментарии, вложения |
| Инженер | Исполнение и техконтроль | Обновление своих задач, комментарии, файлы, перевод на проверку |
| Монтажник | Полевое исполнение | Просмотр своих задач, комментарии, фото, статус выполнения |
| Наблюдатель | Руководство/смежные отделы | Только просмотр |

## Принцип

Доступ выдаётся по проектам. Пользователь должен видеть только те проекты, в которых он реально участвует.
