# Telegram MVP

## Цель

Telegram используется как быстрый полевой интерфейс для монтажников, но не как основная база задач.

Источник истины: OpenProject.

## MVP-функции

1. Уведомление в группу о новой задаче.
2. Напоминание о задачах на сегодня.
3. Напоминание о просроченных задачах.
4. Команда `/done TASK_ID` — перевести задачу на проверку или закрыть.
5. Команда `/comment TASK_ID текст` — добавить комментарий.
6. Загрузка фото к задаче через команду `/photo TASK_ID`.

## Пример сообщения

```text
Новая задача
Проект: Монтаж GPS
Задача: [34ABC123] Установить GPS-трекер
Ответственный: Иван
Срок: 15.06.2026
Приоритет: Высокий
Ссылка: http://localhost:8080/work_packages/123
```

## Техническая схема

```text
OpenProject webhook -> crm_tsm telegram_bot -> Telegram group
Telegram group -> crm_tsm telegram_bot -> OpenProject API
```
